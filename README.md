# vim 配置

开箱即用的 vim 配置，目录结构：
- init.vim 配置入口文件
- autoload 自动加载目录
- config 配置目录
    - 01-basic.vim 基础配置
    - 02-plugin.vim 插件配置
    - 03-advanced.vim 高级配置
